# AI-Based Interview Monitoring & Evaluation System

Real-time interview monitoring system using multi-modal AI for speech transcription, vocal emotion analysis, and offline speaker detection.

## 📂 Project Structure

* `app.py`: Main Flask server.
* `requirements.txt`: Python dependencies list.
* `test_pyannote.py`: Network testing script.
* `templates/`: Web interface folder.
* `templates/index.html`: Web dashboard UI.
* `src/`: Local development scripts.
